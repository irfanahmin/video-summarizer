Video to Notes: Intelligent Lecture Summarization System
Designed an AI-powered system that transforms educational videos into concise, structured notes using state-of-the-art NLP and speech-to-text models.

Implemented automatic speech recognition to convert lecture audio into accurate text using pre-trained models.

Applied text summarization techniques (extractive and abstractive) to generate clear, topic-based summaries.

Enables efficient content review and reduces manual note-taking effort for students and professionals
